# Abstract V1

This project tests whether claim-level uncertainty signals derived from a model's internal activations can outperform structured self-reports for legal contract analysis. We compare three methods on a single open-weights base model, Meta Llama 3.1 8B Instruct: direct self-report by the model, an external GPT-5.4 decomposition baseline applied to the model's answers, and linear probes trained on Llama activations with both sparse-autoencoder features and raw residual representations. The dataset is a hybrid collection of MAUD-style merger-agreement questions and CUAD-derived non-merger contract-review questions, with expert claim-level labels for correctness and, where reliable, load-bearingness. The primary hypothesis is that probe-derived uncertainty will better track expert-annotated claim correctness than prompting-only baselines, especially under cross-distribution shift. A strong null result is equally informative: if cached prompting baselines match or exceed probe quality, the experiment will show that substantial interpretability infrastructure does not yet buy practical claim-level uncertainty decomposition in this domain. The study is designed as a bounded proof of concept with frozen prompts, contract-level holdouts, bootstrap confidence intervals, and an explicit reliability gate on load-bearing annotations.
